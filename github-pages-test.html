GitHub Pages 信息

GitHub Pages 通常需要几分钟来部署更新。

如果产品详情页返回404，可能是：

  1. 文件不存在于仓库中
  2. 文件名或路径错误
  3. GitHub Pages 还在部署中
  4. 需要配置自定义404页面
