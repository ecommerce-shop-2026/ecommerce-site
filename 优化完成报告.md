## 使用指南

### 快速开始
```bash
# 1. 运行主自动化脚本
chmod +x automate.sh
./automate.sh

# 2. 选择要执行的操作
#   1) 运行测试
#   2) 运行备份
#   3) 运行监控
#   4) 运行部署
#   5) 完整流程
```

### 定时任务配置
```bash
# 编辑 crontab
crontab -e

# 添加以下任务
0 1 * * * cd /path/to/project && ./run-backup.sh
*/30 * * * * cd /path/to/project && ./monitor-site.sh
0 2 * * * cd /path/to/project && ./run-tests.sh
```

### 部署到生产环境
```bash
# 1. 安装 CLI 工具
npm i -g vercel netlify-cli

# 2. 登录
vercel login
netlify login

# 3. 部署
./deploy-all.sh
```
