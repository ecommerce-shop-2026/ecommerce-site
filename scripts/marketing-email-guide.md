# ShopEasy 邮件营销方案

## 1. 邮件收集

### 收集方式
- **首页弹窗**: 进入网站5秒后弹出 "Get 10% OFF your first order! Enter your email →"
- **结账页**: 下单时自动收集邮箱
- **博客订阅**: 文章底部 "Subscribe for exclusive deals"
- **退出意图**: 鼠标移出页面时弹窗

### 工具推荐（免费层级）
- **Mailchimp**: 免费2000联系人，每月10000封
- **Brevo (Sendinblue)**: 免费300封/天
- **ConvertKit**: 免费1000联系人

## 2. 邮件序列

```

Welcome! Use code WELCOME10 for 10% off your first order.

👉 Shop Now

Best sellers:
- Wireless Headphones → [link]
- Smart Watch → [link]
- Portable Blender → [link]

Happy shopping!
ShopEasy Team
```

## 3. 最佳发送时间

- 打开率最高: 周二-周四 10am-2pm (美东时间)
- 避免: 周一早上、周五下午、周末
- 频率: 每周1-2封（不过度打扰）
