## 🔗 自定义域名配置（可选）

如果你有自己的域名，可以按照以下步骤配置：

### 步骤1: 购买域名
- 推荐域名注册商: GoDaddy, Namecheap, Google Domains
- 建议域名: shopeasy.com, shop-easy.com, myshopeasy.com

### 步骤2: DNS配置
```
CNAME记录:
名称: www (或 @)
值: ecommerce-shop-2026.github.io
TTL: 3600
```

### 步骤3: GitHub Pages配置
1. 在仓库设置中启用GitHub Pages
2. 输入自定义域名
3. 启用HTTPS强制重定向

### 步骤4: 验证配置
```bash
# 检查DNS解析
nslookup www.yourdomain.com
nslookup yourdomain.com
```
